## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   bun install
   ```
3. Set the environment variables:
   ```
   DB_NAME=
   DB_HOST=
   DB_PORT=
   DB_USER=
   DB_PASSWORD=
   ```
4. Run the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   bun dev
   ```
